Retail debit card usage in Iceland.
-----------------------------------

Description
~~~~~~~~~~~

Monthly retail debit card usage in Iceland (million ISK). January 2000 -
August 2013.

Format
~~~~~~

Monthly time series of class ``ts``.

Source
~~~~~~

Statistics Iceland. https://data.is/HFzL69

Examples
~~~~~~~~

::


   autoplot(debitcards)
