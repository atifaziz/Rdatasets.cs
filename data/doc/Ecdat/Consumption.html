ConsumptionR Documentation

Quarterly Data on Consumption and Expenditure

Description

quarterly observations from 1947-1 to 1996-4

number of observations : 200

observation : country

country : Canada

Usage

data(Consumption)

Format

A time serie containing :

yd

personal disposable income, 1986 dollars

ce

personal consumption expenditure, 1986 dollars

References

Davidson, R. and James G. MacKinnon (2004) Econometric Theory and Methods, New York, Oxford University Press, http://www.econ.queensu.ca/ETM/, chapter 1, 3, 4, 6, 9, 10, 14 and 15.
